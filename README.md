# Hasiru Haadi 🌿

A beautiful, modern website for an online plant shop built with HTML and CSS.

## Overview

**Hasiru Haadi** is a responsive e-commerce platform dedicated to selling indoor and outdoor plants. The website features a clean, nature-inspired design with multiple pages to showcase products, provide information, and facilitate customer engagement.

## Features

✨ **Multi-page Website**
- **Home Page** (`index.html`) - Welcoming landing page with featured content
- **Shop** (`shop.html`) - Product catalog and browsing
- **Gallery** (`gallery.html`) - Visual showcase of plants and products
- **Product Pages** - Individual product detail pages:
  - Little Gem (`product-little-gem.html`)
  - Moss Rose (`product-moss-rose.html`)
  - Snake Plant (`product-snake-plant.html`)
- **About** (`about.html`) - Company information and story
- **Contact** (`contact.html`) - Customer communication
- **Thank You Pages** - Order confirmation pages

🎨 **Design**
- Responsive and modern CSS styling
- Nature-inspired color palette
- User-friendly navigation
- Professional branding with favicon

🛍️ **E-commerce Ready**
- Product browsing and details
- Purchase flow with order confirmation
- Contact and inquiry forms

## Getting Started

### View Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/spandana-builds/hasiru-haadi.git
   cd hasiru-haadi
   ```

2. Open `index.html` in your web browser:
   ```bash
   open index.html
   ```
   Or simply double-click the `index.html` file.

### File Organization

- **HTML Files**: All page content is in HTML files in the root directory
- **CSS Files**: Stylesheets are located in the `style/` directory
- **Images**: Plant images and media assets are in the `images/` directory
